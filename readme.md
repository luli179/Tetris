# a Tetris game

## Require

- linux
- CMake
- g++

## Run

1. Navigate to  project directory

   ```
   cd tetris
   ```

   

2. Create and navigate to the build directory

   ```
   mkdir build
   cd build
   ```

   

3. Run CMake

   ```
   cmake ..
   ```

   

4. Compile the project

   ```
   make -j12
   ```

   

5. Running the Executable

   ```
   ./tetris
   ```


## OR

use run.sh

```
./project directory/run.sh
```
