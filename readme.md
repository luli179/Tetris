# a Tetris game

## Require

- g++
- CMake

## Run

1. Navigate to  project directory

   ```
   cd tetris
   ```

   

2. Create and navigate to the build directory

   ```
   mkdir build
   cd build
   ```

   

3. Run CMake

   ```
   cmake ..
   ```

   

4. Compile the project

   ```
   make -j14
   ```

   

5. Running the Executable

   ```
   ./tetris
   ```
